    Ready4work

    Ready racked storage system.

    What is Ready4Work?

    Ready4Work is a racking system which is fitted before it arrives at the dealer.

    What are the benefits to you?

    • Reduction in leadtime (the vehicle arrives at the dealer Ready4Work)
    • Competitively priced
    • Covered by the same 4yr / 100k mile warranty as your new Renault LCV
    • Can be included in a finance agreement

    System 1

    Has been designed to provide professionals with a durable racking system suitable for the storage of a wide range of tools, components and accessories.

    • Two part anti-slip phenolic coated floor
    • Load and crash tested extruded aluminium floor rails to give maximum strength
    • 10mm plywood base
    • No drilling into vehicle structure
    • Brackets connect to the existing load lashing points
    • Racking manufactured from high strength Dual-Phase steel
    • Lighter and stronger than normal mild steels, helping to maximise payload
    • Durability tested to an equivalent of 65,000 miles of normal road use
    • Weight : 86kg*

    Specialist Storage System

    Designed for the more specialised professional requiring an improved system for the storage of tools and service equipment.

    Construction

    Rear right hand unit includes the following above Storage System 1:

    Telescopic drawer unit consisting of:

    •  2 x (500mm (w) x 75mm (h) x 355mm (d)) drawers
    • 1 x (500mm (w) x 150mm (h) x 355mm (d)) drawer
    • Adjacent to drawer unit is a 500mm(w) x 75mm (h) x 355mm (d) shelf Left hand unit and tool panel assembly unit are as per Storage System 1
    • Weight : 102.4kg*
    * Needs to be subtracted from the payload
